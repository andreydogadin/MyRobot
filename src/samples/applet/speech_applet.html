Speech Applet Demo

This applet has been tested with native VMs on:
  • Internet Explorer 5.5 with Microsoft VM for Java Version 5.0 - the ServicePack 2 for IE 5.5 upgrades the VM to version 5.
  • Netscape 4.7 - Netscape will ask you whether you want to grant permissions to use native code, which you should do if you want to run the applet.
In addition, you must
  • Install CloudGarden's TalkingJava SDK 1.6.0 or better
  • Install the JSAPI for applets, using the installer, or copy the cgjsapiNNN.dll (eg. cgjsapi160.dll) and cgjsapi.jar files to specific locations - as described here.
Alternatively, if the Java Plugin is used to run Java applets (on Netscape 6 & 7 or IE 6, for example), you must
  • install cloudgarden's JSAPI files in the JRE being used by the plugin
  • modify the java policy file as described here
 Type text in the space and press the button to hear it spoken.
